
# Project Title
Eventraze

A brief description of what this project does and who it's for
Eventraze adalah platform lengkap untuk pecinta event! Cari event seru di sekitarmu, promosikan acara kamu,sekaligus katalog event komprehensif. Semua bisa dilakukan dalam satu aplikasi ini dengan mudah, cepat, dan profesional.

Eventraze, one stop platform for every event experience!

## FAQ

1. Apa itu aplikasi Eventraze? 

Answer 1 > Eventraze adalah platform lengkap untuk pencinta event! Cari event seru di sekitarmu dan promosikan acara kamu. Semua bisa dilakukan dalam satu aplikasi Eventraze dengan mudah, cepat, dan proposional. Eventraze, one stop platform for every event experience!

2. Bagaimana cara saya mencari event tertentu? 

Answer 2 > Pilih kategori yang sesuai dengan event yang ingin dicari di halaman beranda

3. Informasi apa saja yang bisa saya lihat di katalog event? 

Answer 3 > Ada banyak hal yang bisa dilihat seperti, nama event, tanggal pelaksanaan, lokasi pelaksanaan, deskripsi singkat tentang event, harga, contact person penyelenggara, benefit yang didapat dari event, dsb.

4. Apakah event yang ditampilkan selalu up-to-date? 

Answer 4 > Ya, Eventraze akan selalu memberikan event-event yang up-to-date

5. Apakah aplikasi ini membutuhkan oneksi internet saat digunakan?

Answer 5 > Ya, supaya anda bisa mengakses semua event-event yang up-to-date

6. Mengapa aplikasi saya berjalan lambat atau mengalami crash?

Answer 6 > Jika terjadi hal tersebut anda dapat melakukan beberapa hal yakni melakukan perbaruan aplikasi, clear cache, atau spesifikasi minimum perangkat

7. Bagaiman jika saya menemukan event yang sudah lewat atau tidak valid? 

Answer 7 > Anda bisa melakukan pelaporan lewat aplikasi ini di bagian bantuan. dengan cara menyampaikan masalah apa yang sedang anda temui di aplikasi Eventraze
